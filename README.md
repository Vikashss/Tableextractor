# Universal PDF Table Extractor

A Streamlit app that converts PDF tables to Excel using the provided universal extractor.

## Features
- Supports digital PDFs
- Supports scanned PDFs with OCR
- Supports mixed PDFs
- Exports all extracted tables to Excel
- Streamlit UI for upload and download

## Files
- `pdf_to_excel_universal.py` → core extraction script
- `app.py` → Streamlit wrapper
